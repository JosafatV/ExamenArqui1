# How valgrind works, in summary:

compile your testFile.cpp to get testFile.o and testFile and run: 

	valgrind --tool=desiredTool ./testFile

Recommended:
	run program with optimization (or as intended)
	compile program in debug mode (flag: -g)
		gcc -o testFile -g testFile.c





# Reading cachegrind
	valgrind.org/docs/manual/cg-manual.html

In summary:

I: instrutions lookups
I1: L1 instruction cache
LLi: last level instruction memory

D: data lookups
D1: L1 data cache
LLd: last level data memory

LL: overall last level accesses



# Specific commands used in this project

Cache grind command
	valgrind --tool=cachegrind ./testFile
